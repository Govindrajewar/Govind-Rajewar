# Govind Rajewar — Portfolio

Personal portfolio website built with React.js, showcasing my experience, skills, and projects as a Full Stack Developer.

**Live:** https://govindrajewar.github.io/Govind-Rajewar

---

## About

Full Stack Developer with 3.6+ years of experience building scalable web applications and REST APIs. Primary stack is Python (Django, FastAPI) on the backend and React.js on the frontend, with production deployments on AWS.

---

## Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React 19, React Router v7 |
| Icons | react-icons v5 |
| Styling | Plain CSS with CSS variables (Midnight Navy theme) |
| Fonts | Inter, Space Grotesk, JetBrains Mono (Google Fonts) |
| Deployment | GitHub Pages via gh-pages |

---

## Pages

| Route | Description |
|---|---|
| `/` | Hero section — typewriter role animation, stats row, CTA buttons |
| `/experience` | Vertical timeline — UrbanMatch Tech & G+D MS India with tech tags |
| `/skills` | 8-category skills grid covering Languages, Frontend, Backend, DB, Cloud, Messaging, Auth, and Tools |
| `/projects` | Featured projects with tech tags and Live Demo / Source Code links |
| `/contact` | Two-column layout — contact info panel + FormSubmit-powered form |

---

## Running Locally

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Deploying to GitHub Pages

```bash
npm run deploy
```

Builds the app and pushes to the `gh-pages` branch automatically.
